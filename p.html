プログラミング班

論理的な思考とコーディングの力で、みんなの課題を解決するデベロッパー。

活動理念・目的

プログラミング班では、「自らの手でアイデアを具現化する」ことを目的に活動しています。最新のWeb言語や開発フレームワークを学び、学内外を快適にするアプリケーションの開発やハッカソン、プログラミングコンテストの参加に向けたチーム活動に力を入れています。

Webアプリ開発

前商祭（文化祭）の特設アプリや、生徒会と連携したデジタルシステムなど、学校行事をテックの力で便利にする活動を行っています。フロントエンドだけでなく、サーバーサイド、データベース設計まで一貫して学習しています。

自動化ツール & ロボット制作

各班や部活動のタスク管理、連絡作業を爆速にするため、DiscordボットやLINEボットなどをPythonを用いて内製。作業時間の大幅なカットに貢献しています。

ハッカソンへの挑戦

企業や大学が主催する、短期間で作品を作る「ハッカソン」へ積極的にエントリー。他校の学生や現役エンジニアとの交流を通じて、技術力の標準を社会に合わせる取り組みをしています。

メインで学習・使用している技術

HTML5 / CSS3 JavaScript (ES6+) React / Next.js Python Git & GitHub
